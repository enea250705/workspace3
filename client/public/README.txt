Per visualizzare correttamente il logo:

1. Aggiungi un file chiamato "logo.png" in questa cartella (client/public/)
2. Il logo dovrebbe essere un'immagine PNG con dimensioni appropriate (consigliato 512x512 px o simili)
3. Assicurati che il file sia nominato esattamente "logo.png" (tutto minuscolo)

Una volta aggiunto il logo, riavvia il server di sviluppo con:
- cd client
- npm run dev
